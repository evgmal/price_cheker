# Создание иконок для PWA

Для корректной работы PWA приложения необходимы иконки двух размеров:
- `icon-192.png` - 192x192 пикселей
- `icon-512.png` - 512x512 пикселей

## Способы создания иконок

### 1. Онлайн генераторы (Рекомендуется)

#### PWA Asset Generator
1. Перейдите на https://www.pwabuilder.com/imageGenerator
2. Загрузите ваше изображение (минимум 512x512 пикселей)
3. Скачайте сгенерированные иконки
4. Переименуйте файлы в `icon-192.png` и `icon-512.png`

#### Favicon.io
1. Перейдите на https://favicon.io/
2. Выберите "PNG to Favicon" или "Text to Favicon"
3. Создайте иконку
4. Скачайте и используйте нужные размеры

### 2. Графические редакторы

#### Photoshop / GIMP
1. Создайте новый файл 512x512 пикселей
2. Нарисуйте или вставьте ваш логотип
3. Экспортируйте как PNG
4. Создайте копию и измените размер до 192x192

#### Canva
1. Перейдите на https://www.canva.com
2. Создайте дизайн 512x512 пикселей
3. Добавьте текст, иконки, цвета
4. Скачайте как PNG
5. Используйте онлайн инструмент для изменения размера до 192x192

### 3. Быстрый способ - генерация из текста

Создайте простую иконку с помощью HTML5 Canvas:

1. Откройте файл `icon-generator.html` (создан ниже)
2. Измените текст/цвет по желанию
3. Кликните правой кнопкой на иконку
4. Выберите "Сохранить изображение как..."
5. Сохраните как `icon-192.png` и `icon-512.png`

## Требования к иконкам

- **Формат**: PNG с прозрачностью (рекомендуется) или с фоном
- **Размеры**: точно 192x192 и 512x512 пикселей
- **Цвет**: Лучше использовать яркие цвета для лучшей видимости
- **Дизайн**: Простой и узнаваемый, без мелких деталей
- **Безопасная зона**: Оставьте 10% отступы от краёв (важные элементы в центре)

## Рекомендации по дизайну

1. **Простота**: Иконка должна быть понятной даже в маленьком размере
2. **Контраст**: Используйте контрастные цвета для лучшей видимости
3. **Символизм**: Используйте символы, связанные с функцией приложения
4. **Консистентность**: Иконка должна соответствовать общему стилю приложения

## Примеры идей для иконки Прайс Чекера

- Штрих-код
- Ценник
- Корзина с ценником
- Знак рубля/доллара
- Сканер
- Комбинация штрих-кода и цены

## Временное решение

Если нужно быстро протестировать приложение, можно использовать `icon-generator.html` для создания базовой иконки с инициалами "ПЧ" (Прайс Чекер).
