Создание временных иконок

Эта страница поможет создать базовые иконки для тестирования PWA

192x192



512x512



Инструкция:

  1. Нажмите "Скачать обе иконки" или скачайте каждую отдельно
  2. Сохраните файлы как icon-192.png и icon-512.png
  3. Поместите их в папку с проектом
  4. Перезагрузите приложение

Примечание: Это базовые иконки для тестирования. Для production используйте генератор иконок или создайте собственные иконки.
